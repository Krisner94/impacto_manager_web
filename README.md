# Documentação de Progresso do Impacto Manager

  - Professores

- Lembretes de Aniversário

- Inserção de autenticação 
- Autenticação de Dois Fatores (2FA)

### Outros Recursos

- Perfis de Alunos e Professores

- Desenvolvimento de Aplicativo Móvel
